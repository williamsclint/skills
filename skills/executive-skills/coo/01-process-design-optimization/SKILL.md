# Process Design & Optimization

**Persona:** COO
**Classification:** INTERNAL USE ONLY
**Department:** Office of the COO | Operations Excellence

## Description
This skill enables the COO to diagnose operational inefficiencies, redesign end-to-end processes, and drive measurable performance improvements across Wynn Resorts' hospitality and gaming operations. It synthesizes current-state process maps, operational performance data, and standard operating procedures to identify root causes of underperformance and produce actionable redesign recommendations. The output is calibrated to the luxury hospitality operating environment where guest experience is the primary constraint.

## When to Use
- When guest satisfaction scores or NPS decline and root cause analysis of operational processes is required
- When a new service offering or amenity is being designed and process architecture must be defined upfront
- When an existing process is generating excessive cycle time, error rates, or labor cost
- When a property is preparing for a major event or peak season and process readiness must be validated
- When benchmarking reveals that a specific operational metric lags significantly behind luxury peers

## Input Files Expected
| # | Document Name | Purpose |
|---|---------------|---------|
| 1 | Current Process Map — VIP Check-In & Concierge Services | Provides the current-state end-to-end process with step-level cycle times, error rates, and pain points |
| 2 | Operational Performance Metrics — FY2024 | Supplies quantitative performance baselines across all key operational metrics to validate where redesign is most urgent |
| 3 | Standard Operating Procedures — Selected Excerpts | Defines the currently mandated process standards to assess whether performance gaps are execution or design failures |

## Output
A process optimization report including: a current-state diagnostic summary with pain point heat map, root cause analysis for underperforming steps, a proposed future-state process design with expected improvement targets, a prioritized list of quick-win vs structural changes, and a draft implementation roadmap with ownership assignments. Includes updated SOP language where process changes have been finalized.

## Example Prompt
> "Our VIP check-in process is generating complaints and our front desk wait time metric missed target in every quarter of FY2024. Walk through the current process map, identify where the bottlenecks are, and recommend a redesigned process with realistic improvement targets."
